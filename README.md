# play_IDE
messing around with some barebones constructs required in an IDE

## setup instructions:
- install `python 3.11` or higher.
    - [Python Download](https://www.python.org/downloads/)
- clone repo: 
    ```shell
    git clone git@github.com:praisetompane/play_IDE.git
    ```
## run program:

- to start system run:
    ```shell
    python main.py
    ```
